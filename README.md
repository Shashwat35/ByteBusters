# Empty Classroom Finder

A mobile-first web app for IIPS DAVV, Indore, to locate free classrooms. Users can filter by floor, capacity, and facilities, select custom time ranges, view rooms on maps, get AI predictions, and receive notifications for favorites.

## Features
- Google Sign-In authentication
- Real-time classroom availability
- Custom time selection
- Filter by floor, capacity, facilities
- AI-powered predictions for next 30/60/90 mins
- Interactive map of campus
- Save favorites & push notifications

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Firebase (Firestore, Auth, Hosting)
- APIs: Google Maps, Google Calendar
- AI: GPT/Gemini/Claude (NLP for natural queries)

## Setup
1. Clone the repo
2. `npm install`
3. Set up Firebase config
4. `npm start` to run locally
